Lab 7: Project Kickoff

Introduction, Data, Methods

Project Setup

Today in lab, you will work with your team and TA to start tackling your final project based on your selected proposal from Lab 5. To start, have your corresponding author (can change from last time) create a new Quarto Manuscript project in RStudio. This will be the central location to work on the project. Please work through the following steps to get started if you need a refresher on setting this up!

1. Create a new RStudio project

2. Add a .git archive (usethis::use_git())

3. Sync it to Github (usethis::use_github())

4. In your RStudio termnial complete the following:

  1. Type: quarto create project

  2. Use the arrow keys to move down and select Manuscript

  3. Enter directory as . (shorthand for “here”)

  4. Enter the Title for the project (e.g. “ESS 330 Project Proposals”)

  5. Hit Enter to create the project!

5. Change your output-dir to docs in the global yml file.

6. Open the index.md file in the manuscript folder and add all authors to the metadata.

Todays task

In lab today, you will work with your team to start building out your project. This will include:

  1. Drafting your Title

  2. Drafting the introduction, background, and motivation for your project (~900 words)

  • Include a brief overview of the problem you are addressing
  • Include a brief overview of the data you are using
  • Include a brief overview of the methods you are using
  • Include APA citations for any sources you use
  1. Exploring your data
  • Creating some visualizations and descriptions of what data you have, where you got it, and how and if you need to clean and manipulate it for your project.
  1. Identify your preliminary methdods
  • What are you thinking about using to analyze your data to answer your question?
  • Do you have everything you need? What supplimental data might you need?
  • What are some potential challenges you see in the data?
  • What are some potential challenges you see in the methods?
  • How do the selected methods help you get to your final goal?

Please refer to the Project Introduction for specifics on these sections!
